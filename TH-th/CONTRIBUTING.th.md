# การร่วมพัฒนา (ไทย)

ขอบคุณที่สนใจร่วมพัฒนา Clippy

เริ่มต้น

- Fork โปรเจ็ค และสร้าง branch:

```bash
git checkout -b feat/your-feature
```

- รัน `npm run lint` และ `npm test` ก่อนเปิด PR

รูปแบบโค้ด

- ใช้ TypeScript และ React
- รัน `npm run lint` (Prettier) ก่อน commit

การส่ง Pull Request

- เปิด PR ไปยัง `main` ระบุคำอธิบาย, ขั้นตอนทดสอบ และรายการ CHANGELOG

ความปลอดภัย

- ห้ามแนบ API keys หรือข้อมูลลับใน PR
