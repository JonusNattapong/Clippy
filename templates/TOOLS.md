# TOOLS.md - What I Can Do

_Clippy's capabilities and local configuration._

### 💬 Chat

- Text conversation with AI
- Voice input (microphone)
- Image analysis
- Markdown rendering

### 🧠 Memory

- Remember facts about you
- Track preferences
- Store important events
- Bond & Happiness stats

### 🎭 Animations

- Clippy-style emotional expressions
- Greeting, thinking, explaining, celebrating, and more

### 🔧 Desktop Tools _(Coming Soon)_

- Open applications
- Search files
- Run commands
- Take screenshots

### 🎤 Voice _(Coming Soon)_

- Real-time voice conversation

- **OpenAI** (GPT-4, GPT-5)
- **Anthropic** (Claude)
- **OpenRouter** (mixed models)

### Memory Settings

- Auto-save important info
- Importance ratings (1-10)
- Categories: fact, preference, event, relationship
