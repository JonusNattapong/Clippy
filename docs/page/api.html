API / IPC

This document summarizes the IPC messages between the Renderer and Main processes.

Key Channels

Channel Direction Description
clippy_chat_stream_start Renderer → Main Start chat completion stream
clippy_chat_stream_chunk Main → Renderer Stream response chunk
clippy_desktop_tool_execute Renderer → Main Execute desktop command
clippy_web_search Renderer → Main Search the web
clippy_memory_* Both Memory operations

Desktop Commands

Available via /command in chat:

  • /run <cmd> - Run PowerShell command
  • /ls [path] - List directory
  • /read <file> - Read file
  • /search <query> - Search files
  • /sysinfo - System info
  • /ps [limit] - List processes
  • /clipboard - Read clipboard
  • /screenshot - Take screenshot

Web Commands

  • /search <query> - Web search
  • /fetch <url> - Fetch webpage

See src/ipc-messages.ts for full message definitions.
