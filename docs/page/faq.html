FAQ / Troubleshooting

App doesn't start

Check Node version and run npm ci then npm run start. See logs in the console.

API key not working

Make sure your API key is correct. Check the .env file or Settings in the app. For cloud providers, ensure you have credits available.

Ollama not connecting

Ensure Ollama is running: ollama serve. Also make sure a model is pulled: ollama pull llama3.2.

Models not found

Ensure local model paths are configured and Ollama/local services are running.

How to use desktop commands?

Type commands like /ls, /read file.txt, /sysinfo in the chat. See full list in the API docs.

How to enable web search?

Get a Tavily API key from tavily.com and add it in Settings.

Where is user data stored?

Windows: %APPDATA%\Clippy\, macOS: ~/Library/Application Support/Clippy/
