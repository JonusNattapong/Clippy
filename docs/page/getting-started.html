Getting Started

Quick steps to run Clippy locally for development.

Prerequisites

  • Node.js (>=18)
  • npm or pnpm
  • Operating system: Windows/macOS/Linux

Development

git clone https://github.com/JonusNattapong/Clippy.git
cd Clippy
npm ci
cp .env.example .env
# Edit .env to add any API keys (GEMINI_API_KEY, OPENAI_API_KEY, etc.)
npm run start

Local models (optional)

If you want to use local models (recommended for privacy/offline):

  1. Install and run Ollama
  2. Run: ollama serve
  3. Pull a model: ollama pull llama3.2
  4. Configure Ollama in Settings

Troubleshooting

  • If the renderer window is blank, open DevTools via the app menu and check console errors.
  • Ensure Node version meets minimum requirement: node -v.
  • On Windows, run PowerShell as Administrator if installation fails for native modules.
