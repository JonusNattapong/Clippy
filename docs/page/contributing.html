Contributing

Thanks for wanting to contribute! A few guidelines:

  • Run npm run lint to format TypeScript files with Prettier.
  • Create PRs against main and include screenshots if UI changes.
  • Do not add tests unless they provide clear value for the change.
  • Follow the existing code style (see AGENTS.md).

Development workflow

npm ci
npm run lint
npm test
npm run start

Project Structure

  • src/main/ - Electron main process
  • src/renderer/ - React UI
  • src/sharedState.ts - Shared state types
  • AGENTS.md - Developer guidelines

Code Style

  • Files: kebab-case (e.g., ipc-messages.ts)
  • Components: PascalCase (e.g., App.tsx)
  • Use Prettier for formatting
  • TypeScript strict mode
