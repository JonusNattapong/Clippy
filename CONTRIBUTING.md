# CONTRIBUTING (English)

Thank you for contributing to Clippy.

Getting started

- Fork and create a feature branch:

```bash
git checkout -b feat/your-feature
```

- Run `npm run lint` and `npm test` before opening a PR.

Code style

- TypeScript + React
- Run `npm run lint` (Prettier) before commit.

Pull Request process

- Open PR against `main` with description, testing steps, and changelog entry.

Security

- Do not include API keys or secrets in PRs. Report vulnerabilities privately to maintainers.
